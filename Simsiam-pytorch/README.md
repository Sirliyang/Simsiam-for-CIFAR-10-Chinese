# SimSiam-pytorch

[Exploring Simple Siamese Representation Learning](https://arxiv.org/abs/2011.10566)
的简单实现，使用了[SaeedShurrab/SimSiam-pytorch](https://github.com/SaeedShurrab/SimSiam-pytorch)
建立的Simsiam模型，加入了一些中文注释，并且加入了简单的训练过程

# 依赖

主要需要GPU版的pytorch

# 使用

直接运行main.py
